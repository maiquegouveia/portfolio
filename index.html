Sobre Mim

Python 3

Python é uma poderosa linguagem de programação ideal para scripts e desenvolvimento rápido de aplicativos. É usado em desenvolvimento web, computação científica e matemática para interfaces gráficas de usuário de desktop.

C

C pode ser usada em diversos tipos de projeto, como a criação de aplicativos, sistemas operacionais, drivers, entre outros. Trata-se de uma linguagem estruturada com um elevado nível de portabilidade.

Java

Java é uma linguagem de programação de propósito geral, baseada em classes e orientada a objetos, projetada para ter menos dependências de implementação. É uma plataforma de computação para desenvolvimento de aplicativos.

JavaScript

JavaScript é uma linguagem dinâmica que é usada para desenvolvimento web, em aplicativos web, para desenvolvimento de jogos e muito mais. Ele permite implementar recursos dinâmicos em páginas da web que não podem ser feitos apenas com HTML e CSS.

HTML5

HTML5 é uma linguagem de programação que permite modificar a aparência de páginas da web, bem como fazer ajustes em sua aparência. Também costumava estruturar e apresentar conteúdo para a web.

CSS3

CSS3 permite que o designer crie sites ricos em conteúdo e com pouco código. Essa linguagem traz alguns recursos interessantes que tornam a página com boa aparência, simples para o usuário navegar e funciona perfeitamente.

MySql

Um programa flexível e poderoso, o MySQL é o sistema de banco de dados de código aberto mais popular do mundo. O MySQL armazena dados em tabelas compostas de linhas e colunas. Os usuários podem definir, manipular, controlar e consultar dados usando a Structured Query Language, mais comumente conhecida como SQL.

Prototipagem

Figma é um aplicativo de design de interface de usuário e edição de gráficos baseado na web. Você pode usá-lo para fazer todos os tipos de trabalho de design gráfico de sites de wireframes, projetar interfaces de aplicativos móveis, criar protótipos, criar postagens de mídia social e tudo mais.

Eu sou Desenvolvedor Full Stack

Um desenvolvedor Full Stack é capaz de trabalhar com toda a pilha de desenvolvimento de um projeto. Além de conseguir atuar como Front-end e Back-end, esse profissional também pode contribuir em todas as etapas e partes de um sistema, como o servidor e o banco de dados.
